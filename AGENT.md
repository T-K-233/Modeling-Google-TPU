# AGENTS.md

## Repository expectations

- This repository uses `uv` as package and project manager.
- Document new architectural discoveries, fixes existing if necessary, in `docs/`. Make sure that the written markdown document matches with the code implementation.

## General repository overview

Here, we are trying to model Google TPU architecture for education purpose. More information is available in `docs/`. Feel free to follow the links in the documents and query internet for more information. If you have new findings or more correct understanding of how Google TPU works, update the document to reflect that, and fix the code to accurately reflect the actual Google TPU behavior.

There are some assembly programs in `tests/` folder. Those are generated by running kernels on Google TPU and getting compiler dump from the XLA compiler. Those should give understanding of how each instruction behaves.
